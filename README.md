# Windows Icon Animation Engine


- 通过 Windows Desktop Shell 获取 `IShellView` / `IFolderView`
- 枚举桌面图标 PIDL 并读取坐标
- 在内存中保存原始布局
- 规范识别系统回收站，将它放在显示器底部作为接收器
- 其他图标在顶部保持静止，每次随机选择一个图标垂直下落
- 图标先开始下落，回收站稍后启动并移动到垂直落点接住它
- 接住后的最后一帧直接把图标移动到显示器工作区之外，不播放额外隐藏动画
- 接球模式每帧只更新下落图标和回收站，顶部静止图标不重复写入坐标
- 回收站每接住一个图标都会下沉、弹起和抖动
- 最后一个图标被吞入后，回收站蓄力爆炸，所有图标弹射回顶部槽位
- 爆炸后切换到重力碰撞模式：每轮随机打乱图标，以约 450ms 间隔从顶部释放、碰撞并落地回弹
- 重力模式稳定后重新排列到顶部，再切回回收站接球模式持续循环
- 桌面未显示回收站时自动跳过接球模式，只循环重力碰撞；异常识别到多个时安全退出
- 重力碰撞的地面使用显示器工作区底边，图标停在任务栏顶部
- 重力模式每帧只更新已经释放的图标，尚未掉落的顶部图标保持 Shell 原状态
- 动画期间自动关闭“自动排列图标”和“将图标与网格对齐”
- 全局快捷键 `Ctrl+Alt+Q`、控制台关闭或正常结束时恢复原始布局及原来的排列设置

## 运行

dist/windows-icon-animation-engine.exe

```powershell
cargo run --release
```

按全局快捷键 `Ctrl+Alt+Q` 停止动画并恢复图标位置。程序只支持 Windows，且需要在交互式 Explorer 桌面会话中运行。

可先执行只读自检；它会连接 Desktop Shell、读取所有坐标并退出，不会移动图标：

```powershell
cargo run --release -- --check
```
